# Milestone 4 Master: Merchant / Admin Portal

**Budget:** $1,000  
**Timeline:** Mid-Week 6

## Objective

Develop a merchant-facing web portal for order management and operational visibility. Integrate with the client-provided Terminal Bridge to process in-store terminal payments and update POS ticket results.

## 🏢 Admin Features

- **Management Dashboard**: Visual Sales Charts and Popular Items reports.
- **Order Queue**: Live view with Search/Filter and Cancel/Refund processing.
- **Security**: Activity Logs and Optional IP Whitelisting.

## ✅ Milestone 4 Deliverables

- [ ] Merchant / Admin Portal Dashboard
- [ ] Order Queue & Operational Views
- [ ] Authorize.net Payment Visibility & Validation
- [ ] Terminal Bridge Request/Response Integration
- [ ] POS Posting for Bridge Results
