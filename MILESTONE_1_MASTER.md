## 🏛️ System Architecture

The system follows a three-tier architecture:

1. **Frontend**: React Native (Mobile) and Next.js (Web).
2. **Backend**: ASP.NET Core 8 Integration Service (Dapper for SQL Server 2005 compatibility).
3. **Storage**: Legacy `INI_Restaurant` SQL Database + Backend Overlay DB.
