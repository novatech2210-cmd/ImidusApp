## Project Overview

POS Integration Platform connects restaurant POS systems (Toast/TopTop) with mobile and web ordering applications, enabling seamless online ordering without expensive infrastructure upgrades.

## Tech Stack Summary

| Platform | Technology |
|----------|-------------|
| Mobile | React Native 0.73 (iOS + Android) |
| Web | Next.js 14 |
| Backend | .NET 8 Windows Service |
| Database | MS SQL Server 2005 Express |
| Payments | Authorize.net Accept.js |
| Push | Firebase FCM |
