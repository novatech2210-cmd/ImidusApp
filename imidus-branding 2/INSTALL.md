## 1. React Native Mobile App

### 1a. Logo assets

Copy every `.png` from `mobile/src/assets/images/` into:

```
POSMobile/src/assets/images/
```

### 1b. Theme files

Copy the entire `mobile/src/theme/` folder into:

```
POSMobile/src/theme/
```

Then add the path alias in `tsconfig.json` / `babel.config.js`:

```json
"paths": { "@/theme": ["src/theme/index.ts"] }
```

### 1c. Shared components

Copy `mobile/src/components/common/` into:

```
POSMobile/src/components/common/
```

This gives you `LoginScreen.tsx`, `ImidusHeader.tsx`, and `LoyaltyCard.tsx` — fully branded and ready to use.

### 1d. Android colours + styles

```
mobile/android/app/src/main/res/values/colors.xml   → android/app/src/main/res/values/colors.xml
mobile/android/app/src/main/res/values/styles.xml   → android/app/src/main/res/values/styles.xml
mobile/android/app/src/main/res/drawable/splash_background.xml  → android/app/src/main/res/drawable/
mobile/android/app/src/main/res/drawable/splash.png             → android/app/src/main/res/drawable/
```

Then in `AndroidManifest.xml`, set `android:theme="@style/SplashTheme"` on the launch activity and switch to `@style/AppTheme` after the splash.

### 1e. iOS splash screen

```
mobile/ios/LaunchScreen.storyboard  → ios/POSMobile/LaunchScreen.storyboard
```

Add `splash.png` to `ios/POSMobile/Images.xcassets/Splash.imageset/`.  
Set background colour to `#1E5AA8` in the storyboard's view.

### 1f. App icon (both platforms)

Use `app-icon-512.png` as the source:

- **iOS**: Xcode → Images.xcassets → AppIcon → drag & drop 512×512, generate all sizes
- **Android**: Android Studio → Image Asset → choose `app-icon-512.png` as foreground, background `#000000`

## 2. Web Ordering Site (Next.js)

### 2a. Logo assets

Copy all `.png` files to:

```
web-ordering/public/images/
```

### 2b. Global CSS

```
web/src/styles/globals.css  → web-ordering/src/styles/globals.css
                               (or app/globals.css for App Router)
```

Import at the top of `_app.tsx` or `layout.tsx`:

```tsx
import "@/styles/globals.css";
```

### 2c. Header & Footer

```
web/src/components/layout/SiteHeader.tsx          → src/components/layout/
web/src/components/layout/SiteHeader.module.css   → src/components/layout/
web/src/components/layout/SiteFooter.tsx          → src/components/layout/
```

Use in your `_app.tsx` / root layout:

```tsx
import SiteHeader from "@/components/layout/SiteHeader";
import SiteFooter from "@/components/layout/SiteFooter";
```

### 2d. next/image domains

In `next.config.js`, add the S3 bucket domain if loading logos remotely:

```js
images: {
  domains: ['inirestaurant.s3.amazonaws.com'],
}
```

## 4. Backend .NET Service

### 4a. Brand constants

```
backend/ImidusConstants.cs  → IntegrationService.API/Constants/ImidusConstants.cs
```

Reference `ImidusConstants.BrandGold`, `ImidusConstants.PushTitlePrefix`, etc. across the service.

### 4b. Email template

```
backend/BaseEmailTemplate.html  → IntegrationService.API/Templates/Email/BaseEmailTemplate.html
```

Replace `{{PLACEHOLDERS}}` in `EmailService.cs` using `string.Replace()` or Handlebars.Net.

Populate logo URLs from S3:

```csharp
var logoUrl = $"https://{ImidusConstants.S3BucketName}.s3.amazonaws.com{ImidusConstants.S3LogoBlueBanner}";
```
